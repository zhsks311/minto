# 전역 용어집 메인 다이얼로그 구현 계획

## 목표

- 전역 용어집 관리 UI를 설정 화면에서 메인 화면 다이얼로그로 이동한다.
- PRD와 Pencil 디자인에 있는 전역 용어집 관리 범위만 구현한다.
- 새 회의, 파일 가져오기, 다시 요약의 회의별 용어 선택 흐름은 유지한다.

## 단계

1. 기존 설정 용어집 UI를 재사용 가능한 관리 view와 sheet로 분리한다.
   - 검증: 기존 저장, 수정, 삭제, 후보 승인 동작 코드가 유지되는지 확인한다.
2. 메인 회의 목록 헤더에 `용어집` 버튼을 추가하고 sheet를 연결한다.
   - 검증: 넓은 헤더와 좁은 헤더 양쪽에서 버튼이 렌더링되는지 확인한다.
3. 설정 화면에서 전역 용어집 섹션을 제거한다.
   - 검증: `SettingsView`에 `GlossarySettingsSection` 호출이 남지 않는지 확인한다.
4. 지정된 검증을 수행한다.
   - 검증: glossary 테스트, `./scripts/dev.sh build`, 앱 실행 확인.
