핵심 판단

Decision 기본 STT는 아직 WhisperKit large-v3 626MB 유지
docs/stt-poc-summary.md

120초 meeting sample 기준 WhisperKit final global CER는 26.0%였다. sherpa-onnx는 훨씬 빠르지만 같은 sample에서 33.3%~34.5%로 정확도가 낮아 기본값 교체 근거가 부족하다.

Measured Streaming benchmark는 main-ready 수동 테스트로 추가됨
StreamingChunkBenchmarkTests.swift

RUN_STT_TESTS=1와 RUN_STREAMING_BENCH=1 없이는 실행되지 않는다. 기본 테스트에서는 discovery만 되고 바로 return하므로 일반 test suite를 느리게 만들지 않는다.

Plan 전사 품질 문제는 threshold 하향보다 stop drain과 저장 normalizer가 먼저
docs/transcript-reliability-plan.md

짧은 발화 누락과 저장 transcript의 30초 chunk 느낌은 단순 threshold 조정으로 해결하면 phantom transcript 위험이 커진다. 우선 VAD pending buffer drain, final-empty preview 상태 안정화, 저장용 transcript block 정규화를 순서대로 구현하는 계획이다.
