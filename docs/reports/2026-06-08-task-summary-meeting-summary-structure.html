Summary

회의 요약을 단순 본문에서 결정사항, 할 일, 미해결 질문 중심의 업무 산출물로 확장했다. 새 구조는 저장 JSON, 회의 상세 UI, 검색, 관련 문서 쿼리, Markdown 내보내기에 모두 반영됐고 전체 테스트와 개발 빌드가 통과했다.

업데이트된 기능

회의 요약 구조화

회의 종료 후 생성되는 요약이 기존 리드 요약과 본문 정리에서 업무 결과 중심 구조로 확장됐다.

  • 결정사항: 회의에서 합의된 내용
  • 할 일: 후속 작업, 담당자, 기한
  • 미해결 질문: 추가 확인이 필요한 내용

회의 상세 화면

요약 탭에서 사용자가 회의 후 바로 확인해야 하는 항목을 먼저 볼 수 있게 바뀌었다.

  • 요약 아래에 결정사항/할 일/미해결 질문 블록 표시
  • 각 블록별 복사 버튼 제공
  • 기존 회의 내용 정리와 전사 탭은 유지

검색과 관련 문서 조회

회의 검색과 관련 문서 검색이 새 구조화 필드를 우선 활용하도록 개선됐다.

  • 결정사항, 할 일, 담당자, 기한, 질문 검색 가능
  • 검색 결과에 결정/할 일/질문 매칭 이유 표시
  • 관련 문서 query에 업무 결과 항목 우선 반영

Markdown 내보내기

복사나 파일 저장 시 회의록이 더 실행 가능한 문서 형태로 나가도록 바뀌었다.

  • ## 결정사항 섹션 추가
  • ## 할 일 체크박스 섹션 추가
  • ## 미해결 질문 섹션 추가
