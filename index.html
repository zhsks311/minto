프라이버시 경계

'로컬'을 뭉뚱그리지 않습니다

어디까지 기기 안이고 어디서부터 밖인지, 경계를 정확히 긋습니다. 가장 중요한 원칙은 하나 — 녹음된 회의 음성은 어떤 이유로도 외부 서버로 전송되지 않습니다.

안

기기 안에서만

음성 캡처, 음성→텍스트 전사(STT), 회의 검색 임베딩. STT 모델만 최초 1회 내려받고, 이후 전사는 오프라인으로 돕니다.

음성은 안 나감
밖

켰을 때만, 텍스트만

AI 교정·요약·검색 답변에 클라우드 LLM을 직접 선택한 경우에만 나갑니다. 음성이 아니라 텍스트이고, 검색 답변은 관련 근거 조각만 보냅니다.

사용자 선택
전부

완전 로컬도 가능

LLM마저 Ollama 같은 로컬 모델로 돌리면 교정·요약·검색까지 전 과정이 오프라인입니다. 토큰·프롬프트·전사 원문은 로그에도 남지 않습니다.

전 과정 오프라인
왜 다른가

업로드하는 회의록과 반대 방향

대부분의 클라우드 회의록 앱은 음성을 자사 서버로 올리고 매달 구독료를 받습니다. Minto2는 내 하드웨어로 실행하고, 음성은 기기에 둡니다.

일반 클라우드 회의록 앱Minto2
음성 처리서버로 업로드해 전사기기 안에서 전사 (WhisperKit + Metal)
오프라인불가음성 전사는 오프라인 동작
비용월 구독앱 자체는 구독 없음 — 내 하드웨어로 실행
한국어영어 우선, 한국어는 들쭉날쭉한국어 회의 특화 — 문맥·용어집 교정
AI 요약제공사 모델 고정, 클라우드 강제로컬 LLM 또는 원하는 클라우드 LLM 선택
내 기록제공사 클라우드에 종속내 디스크에 JSON으로 저장
기능

받아 적고, 정리하고, 다시 찾기

회의 한 번이 검색 가능한 지식이 되기까지 필요한 것들.

🎙

실시간 전사

마이크와 시스템 오디오를 함께 캡처해 화상회의 상대방 목소리까지 받아 적습니다. 떠 있는 오버레이로 흐름을 확인합니다.

📝

구조화된 회의록

요약·목차·결정사항·할 일·미해결 질문으로 자동 정리합니다.

🗣

화자분리

발화 구간에 화자 라벨을 붙이고 실명 지정. 보이스프린트로 회의 간 같은 화자를 매칭합니다.

📂

파일로 회의록

기존 음성·영상 파일을 넣어 사후 전사. 일반 회의와 똑같이 저장됩니다.

🔎

검색 & 답변

저장된 회의에 질문하면 출처 회의·시간과 함께 근거 기반으로 답합니다. 로컬 임베딩 semantic search.

📖

용어집

전문용어·고유명사를 등록해 전사 교정 품질을 높입니다. 전역 용어집과 회의별 용어를 함께 씁니다.

🔗

업무 도구 연동

Confluence·Notion 문서 조회, Markdown·Confluence로 내보내기.

🧰

메뉴바 상주

회의 시작과 종료를 메뉴바에서 바로 제어합니다.

🧠

LLM 선택권

OpenAI·Gemini·Claude·OpenRouter, 로그인 계정, 로컬 Ollama, Claude Code CLI까지 골라 씁니다.

동작 방식

세 가지 핵심 흐름

01

실시간 회의 기록

  1. 새 회의를 시작합니다.
  2. 주제·이번 회의 용어·참고 문서를 선택적으로 입력합니다.
  3. 마이크·시스템 사운드·혼합 입력을 전사합니다.
  4. 종료 시 요약·결정사항·할 일까지 정리해 저장합니다.
02

파일로 회의록

  1. 음성 또는 영상 파일을 넣습니다.
  2. 16kHz mono PCM으로 변환합니다.
  3. 기존 전사·교정·요약 파이프라인을 재사용합니다.
  4. 일반 회의와 동일하게 목록에 저장됩니다.
03

검색과 답변

  1. 회의 목록에서 키워드나 질문을 입력합니다.
  2. 요약·전사·결정사항을 검색합니다(+semantic).
  3. 답변 생성 시 근거 chunk만 LLM에 전달합니다.
  4. 출처 회의·섹션·시간과 함께 답을 보여줍니다.
